GENERAL NOTES

  • Select the appropriate build type and platform from the BASIS Web site at www.basis.com/products/bbj/download.html. Make sure BBj®, Barista®, and AddonSoftware are all checked in the Optional Files section of the page. Refer to the documentation for BBj and/or Barista when installing those products.
  • The default installation path for AddonSoftware is the apps/aon/directory inside the home directory of your BBj/Barista installation. All of the Addon files needed to update the Barista data dictionary and create/run forms, as well as standalone program files (for reports, publics, updates and utilities), data files, etc., can be found in this directory tree.
  • The AddonSoftware installation comes with a demo database. Some of these files are pre-populated and are required to run Addon. Others simply contain demo data for your convenience and may be cleared and/or deleted since Addon defines a file on the first attempt to open it if the file doesn't exist. The following tables should not be deleted, but may be customized with your own data:
